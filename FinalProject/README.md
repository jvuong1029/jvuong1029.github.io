## Keyboard controls:

-   Left key = "."
-   Right key = "-"
-   Down key = " "
    -   One space to complete a character
    -   Two spaces to complate a word
    -   Three spaces for a surprise (or spam spaces without any input)
-   Backspack = delete previous input
